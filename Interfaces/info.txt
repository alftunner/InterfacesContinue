1. Интерфейсные свойства и индексаторы
Свойства - аналог полей в классах. Могут быть: для чтения и записи, только для чтения, и только для записи.
interface IPerson
{
    string LastName { get; set; } - Свойство в интерфейсе - это НЕ автосвойство!
}
Индексатор - это специальный вид свойств.

2. Наследование интерфейсов:
- Интерфейс может наследоваться от других интерфейсов.
- Если класс реализует интерфейс, который в свою очередь наследует другой интерфейс, то в этом классе должны быть реализованы
методы всех интерфейсов по иерархии.

3. Проблема сокрытия имён при наследовании интерфейсов
Когда класс реализует несколько интерфейсов, а у этих интерфейсов есть методы с одинаковым названием и сигнатурой.
